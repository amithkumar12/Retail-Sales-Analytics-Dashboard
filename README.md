# 📊 Retail Sales Analytics Dashboard

An end-to-end Data Analytics project built using **MySQL** and **Power BI**. This project demonstrates the complete workflow from importing raw retail sales data into MySQL, performing data cleaning and transformation using SQL, and building an interactive Power BI dashboard for business insights.

---

## 🚀 Project Overview

The dashboard provides an interactive analysis of retail sales performance using the Sample Superstore dataset.

### Dashboard Highlights

- 📈 Total Sales Analysis
- 📦 Total Orders
- 💰 Total Profit
- 👥 Total Customers
- 📊 Sales by Category
- 🌍 Sales by Region
- 🥧 Sales by Segment
- 📅 Monthly Sales Trend
- 🏆 Top 10 Products
- 👤 Top 10 Customers
- 🎛 Interactive Slicers

## 📂 Project Workflow

### 1. Database Creation
- Created a MySQL database.
- Designed normalized tables for Customers, Products, Orders, and Staging.

### 2. Data Import
- Imported the Sample Superstore dataset into MySQL.
- Validated imported records.

### 3. Data Transformation
- Cleaned data using SQL.
- Handled duplicate product IDs.
- Converted date formats.
- Built normalized tables.

### 4. Power BI Development
- Imported the cleaned dataset.
- Used Power Query for transformations.
- Created DAX measures.
- Built an interactive dashboard.

## 📌 Key Business Insights

- Technology generated the highest sales.

- Consumer segment contributed the largest share of sales.
- Sales increased significantly during the final months of the year.
- Top-selling products accounted for a substantial portion of total revenue.
